# Student Attendance Management System

This is a Django-based web application for managing student attendance with role-based access.

## Features
- User login system
- Roles: Admin, Teacher, Student
- Teacher can mark attendance
- Student can view attendance
- Admin can manage users

## Tech Stack
- Python
- Django
- HTML, CSS
- SQLite
